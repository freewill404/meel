# Meel.me
 Email yourself - [meel.me](https://meel.me)

## Install

```bash
cp .env.example .env

# fill in the arrows in the .env file

composer install

php artisan key:generate

php artisan passport:keys
 
php artisan migrate (--seed)
 
npm install
 
npm run dev
```
